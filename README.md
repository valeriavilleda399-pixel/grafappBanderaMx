# grafappBanderaMx
##Historial de Prompts - Práctica 1.4

Genera una interfaz web con 3 archivos (html, css, js) para la "Bandera de México" con franjas para Esperanza (#009900), Unidad (#FFFFFF) y Sangre (#9D0000).

Haz que los colores cambien dinámicamente usando sliders (input range).

Ajusta el diseño para que sea llamativo, con un fondo degradado desenfocado y estilo "glassmorphism".

En la zona central, sustituye el texto por el Escudo Nacional de México oficial.

Agrega al final el texto: "Villeda Hernandez Valeria Abigail 24200103".
